# Superstore Sales Dashboard (Excel)

## Project Overview

This project analyzes retail sales data using Microsoft Excel. The goal was to identify sales performance, profitability, and product trends across different regions and categories.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* Dashboard Design

## Dataset

* Sample Superstore Dataset
* Contains information about sales, profit, categories, regions, and customer segments.

## Dashboard Features

* Sales by Region
* Profit by Category
* Sales by State
* Top 10 Sub-Categories by Sales
* Interactive Filtering using Slicers

## Key Insights

* Identified the highest-performing sales regions.
* Compared profitability across product categories.
* Analyzed top-performing states by revenue.
* Discovered the best-selling product sub-categories.
